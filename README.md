# SocialInteractionSuggester
This app is a user interface app which suggests a social interaction based on a specific time of day.
It uses the decison logic and if statements which displays a suggestion of what you will do or say when you enter a specific time of day.
This app was created in android studio using the kotlin programming language. 
The app has a few simple features which are:
The app logo/image at the very top
The user input:Where you enter the your specific time of day it has a visible hint.
The Click to suggest button: That displays the suggestion after you click it.
The Reset button: It resets everything back to default state when you click it.
